infoporto.odoo Installation
---------------------------

To install infoporto.odoo using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``infoporto.odoo`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        infoporto.odoo
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
